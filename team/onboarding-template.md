# Onboarding Template - Template สำหรับสร้าง Role ใหม่

*ส่วนนี้คือ template มาตรฐานสำหรับสร้าง Role ใหม่*

## Template สำหรับ Role ใหม่

```markdown
# [Role Name] - [Role Title]

## ความหมาย
[ความหมายของชื่อ - ความหมายที่เกี่ยวข้องกับอวกาศ]

## ความสามารถหลัก
- [ความสามารถที่ 1]
- [ความสามารถที่ 2]
- [ความสามารถที่ 3]

## ความรับผิดชอบ
- [ความรับผิดชอบที่ 1]
- [ความรับผิดชอบที่ 2]
- [ความรับผิดชอบที่ 3]

## Prompt/Instruction สำหรับ AI
เมื่อถูกเรียกใช้ใน Role [Role Name]:
- [คำสั่งสำหรับให้ AI ทำงานใน role นี้]
- [ใช้ knowledge-package หรือ resources อื่นๆ เป็นฐาน]
- [เน้นที่สิ่งที่สำคัญ]
- [อธิบายด้วยภาษาไทย]
- [พิจารณา trade-offs และ reasoning]

## ตัวอย่างงานที่ทำได้
- [ตัวอย่างงานที่ 1]
- [ตัวอย่างงานที่ 2]
- [ตัวอย่างงานที่ 3]

## Skill ที่เกี่ยวข้อง
- [skill name ถ้ามี]
```

## ขั้นตอนการสร้าง Role ใหม่

1. วิเคราะห์งานที่ต้องการ
2. ตั้งชื่อ Role ใหม่ (Space Theme)
3. ใช้ template นี้สร้าง Role file
4. สร้าง Preferences file ใหม่ (ใช้ template เดียวกับ preferences/)
5. อัปเดต `team-config.md`
6. บันทึกการ "จ้างพนักงานใหม่" ลงใน `daily/YYYY-MM-DD.md`

## ชื่อ Space Theme ที่แนะนำ

- **Comet** (ดาวหาง) - สำหรับงานที่ต้องความเร็ว
- **Eclipse** (คราสรุ้ง) - สำหรับงานที่ต้องการความลึก
- **Quasar** (ดาวควาซาร์) - สำหรับงานที่ต้องพลังมาก
- **Meteor** (ดาวตก) - สำหรับงานที่ต้องการ impact
- **Pulsar** (ดาวพัลซาร์) - สำหรับงานที่ต้องการ consistency (ใช้ไปแล้ว)
- **Supernova** (ซูเปอร์โนวา) - สำหรับงานที่ต้องการความสว่างมาก
- **Black Hole** (หลุมดำ) - สำหรับงานที่ต้องการความลับ/ความปลอดภัย
- **Nebula** (เนบิวลา) - สำหรับงานที่ต้องการเก็บข้อมูล (ใช้ไปแล้ว)
- **Galaxy** (ดาราจักร) - สำหรับงานที่ต้องการมองภาพใหญ่ (ใช้ไปแล้ว)
- **Constellation** (กลุ่มดาว) - สำหรับงานที่ต้องการ coordination
