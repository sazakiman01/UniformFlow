### Task 2: ปรับปรุงการเลือกสีผ้าให้ง่ายขึ้นด้วย Grouping + Search

**ปัญหา:**
- สีผ้าในแคตตาล็อกมีเยอะมาก (80+ สี)
- User เลือกสีลำบากเพราะไม่มีการจัดกลุ่มหรือค้นหา

**วิธีแก้ (Hybrid Approach):**
- เพิ่ม `colorFamily` metadata ในแต่ละสี (blue, gray, green, red, purple, yellow, orange, pink, brown, light)
- สร้าง function `groupColorsByFamily()` จัดกลุ่มสีตาม color family
- เพิ่ม Search/Filter box ใน Select component (ค้นหาได้ทั้ง: รหัสสี, ชื่อสี, hex code)
- อัปเดต ColorCatalogPicker รองรับ grouping + search
- อัปเดตหน้า stock/items/new รองรับ grouping + search

**ผลลัพธ์:**
- User เลือกสีง่ายขึ้นมากแม้มี 80+ สี
- สามารถค้นหาด้วยพิมพ์รหัสสี, ชื่อสี, หรือ hex code
- เมื่อไม่ค้นหา จะเห็นสีแบบจัดกลุ่มตามกลุ่มสี (🔵 สีน้ำเงิน/ฟ้า, ⚫ สีเทา/ดำ/เนื้อ, 🟢 สีเขียว ฯลฯ)
- ดีต่อระยะยาวเมื่อมีแคตตาล็อกเพิ่มขึ้น (scalable)

**ไฟล์ที่แก้ไข:**
- `src/types/index.ts` (เพิ่ม colorFamily ใน FabricColor)
- `src/data/fabric-catalogs.ts` (เพิ่ม metadata + groupColorsByFamily)
- `src/components/ui/select.tsx` (เพิ่ม search box)
- `src/components/stock/ColorCatalogPicker.tsx` (รองรับ grouping + search)
- `src/app/dashboard/stock/items/new/page.tsx` (รองรับ grouping + search)
